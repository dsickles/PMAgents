# User Rules

## Critical Rules
1. **NO UNAPPROVED EDITS:** You are strictly FORBIDDEN from using `write_to_file`, `replace_file_content`, `multi_replace_file_content`, or any other file modification tool without first:
    a. Describing the exact change you plan to make in the chat.
    b. Asking the user for explicit permission.
    c. Receiving a clear "Yes", "Proceed", or similar confirmation from the user.
    *There are NO exceptions to this rule.*

2. **Plan Before Action:** Always present a plan or summary of changes before attempting to execute them.
