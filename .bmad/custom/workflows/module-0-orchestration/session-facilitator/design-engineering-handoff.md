# Design Engineering Handoff

## Goal
To ensure engineering has everything they need to build the design.

## Steps
1.  **Present**: Ask `UX Designer` to link the mockups.
2.  **Critique**: Ask `Technical Partner` for feasibility issues.
3.  **Resolve**: Facilitate negotiation on hard parts.
4.  **Sign-off**: Generate "Ready for Dev" status.

## Output (Handoff Agreement)
Must include:
1.  **Negotiated Changes**: Exactly what changed from the original design during the session.
2.  **Constraint Acknowledgement**: Explicit acknowledgement of technical constraints by the *UX Designer*.
