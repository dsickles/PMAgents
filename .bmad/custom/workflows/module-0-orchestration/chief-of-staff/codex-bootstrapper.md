# Codex Bootstrapper

## Goal
To populate the 6 core Codex files with initial data so other agents can function.

## Steps
1.  **Mission Check**: Ask for Company Vision/Mission. Write to `company_mission.md`.
2.  **Strategy Check**: Ask for Top 3 Strategic Pillars. Write to `product_strategy.md`.
3.  **Persona Check**: Ask for primary user persona. Write to `user_personas.md`.
4.  **Tech Check**: Ask for primary tech stack. Write to `technical_constraints.md`.
5.  **Brand Check**: Ask for tone of voice. Write to `brand_voice.md`.
6.  **Stakeholder Check**: Ask for key decision maker. Write to `stakeholder_map.md`.

## Output (Full File Content)
"Step 1/6: Mission. I've drafted this based on your input:

```markdown
# Mission
To democratize finance.
```

Save and continue?"
